# 📌 Описание:
#### **S2VNA_CONTROL** это клиентское приложение на C++/Qt для удалённого управления ВАЦ через интерфейс TCP/IP. В основе лежит использование программы S2VNA, разработанной ООО «Планар», которая обеспечивает низкоуровневое взаимодействие с ВАЦ и предоставляет возможность внешним приложениям передавать команды и получать результаты измерений через сокет-интерфейс.

#### Программа подходит для использования в лабораториях, научных и производственных проектах, где требуется автоматизация и дистанционное управление измерительным оборудованием.

---

# 💡 Основной функционал:

#### 🟢 Установление и управление сетевым соединением с ВАЦ.

#### 🟢 Передача SCPI-команд.

#### 🟢 Приём и отображение результатов измерений.

#### 🟢 Графический интерфейс на Qt5.

#### 🟢 Поддержка повторного подключения, логирования и обработки ошибок.
