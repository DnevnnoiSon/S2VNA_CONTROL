## 📝 Описание
Управление ВАЦ осуществляется с помощью программы S2VNA,
разработанной в ООО «Планар», которая обеспечивает низкоуровневый обмен данными с ВАЦ, а также предоставляет возможность <mark style="background:darkwhite">пользовательскому приложению</mark> 
передавать команды и получать результаты измерений через интерфейс Socket для
удаленного управления ВАЦ.
